# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-01-XX

### Initial Release

- Prisma DTO generator for NestJS
- Support for Create, Update, Read DTOs
- Swagger/OpenAPI decorators
- Configurable relation handling (omit, ids, nested)
- Domain-based folder structure
- Enum generation
- Field type heuristics (email, URL, UUID detection)
- Configurable field omission and inclusion
- Custom domain mappings (no hardcoded defaults)
